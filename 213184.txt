КОМАНДИ GIT

1.git init иницијализираме Git во фолдер.
2.git remote add origin https://github.com/BorjanGjorgiev/SI_2023_lab1_213184.git
Го поврзуваме локалниот репозиториум со remote репозиториумот
3.nano README.md креираме фајл
4.git commit -m "...."- испраќаме порака за сработеното.
5.git add - поставуваме фајл на сцена.
6.git status - ја прикажува состојбата на директориумот во кој се наоѓаме.Прикажува промени,кои фајлови се tracked а кои не се.
7.git remote -v  -прикажува дали сме добро синхронизирани со remote repo
8.git branch bug1 -креираме branch bug1
9.git branch - приказ на која гранка се наоѓаме
10.git checkout bug1 - влегуваме во гранката bug1.
11.git push -u origin bug1 - правиме push на комитите креирани локално на некоја гранка во случајов bug1
12.git push --all -правиме push на сите комити без разлика на која гранка се
13.git merge - спојуваме гранка на master
14.git log - дава историја на сите комити кои што ги имаме направено.
15.git rebase - ги префрламе промените од една гранка на друга најчесто на master.
